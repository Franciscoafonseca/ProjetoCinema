# Online Cinema Festival

Projeto desenvolvido para a UC de Engenharia de Software.

## Arquitetura

O projeto segue o Caminho B:

- ASP.NET Core Web API
- Blazor WebAssembly
- Entity Framework Core
- SQLite
- Repository Pattern
- Service Layer

## Como correr o projeto

### API

```bash
cd OnlineCinemaFestival.Api
dotnet restore
dotnet ef database update
dotnet run
```
